Help & Guidelines
Shipment Process
{% if direction == 'OUT' %}
  1. Select the target cluster
  2. Set estimated delivery date
  3. Add FCPs and allocate packages
  4. Collection Centre will confirm receipt
  5. Optional: Mark as distributed to FCPs
{% else %}
  1. Cluster and Collection Centre are auto-set
  2. Set estimated delivery date
  3. Add FCPs and specify return quantities
  4. National Office will confirm receipt
  5. SDSA will mark as posted
{% endif %}
Important Notes
  • At least one FCP must be selected
  • Quantities must be greater than 0
  • Duplicate FCPs are not allowed
    • {% if direction == 'OUT' %}
  • Collection Centre FCPs are excluded
    • {% endif %}
Tip: Use the "Add FCP" button to add more FCPs to the shipment.
